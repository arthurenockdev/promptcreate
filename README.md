# PromptCreate

PromptCreate is an AI-powered web application that helps developers build full-stack applications through natural language conversations. Simply describe what you want to build, and the AI will help you create it!

## Features

- **AI-Powered Development**: Chat with an AI assistant to generate and modify code
- **Live Code Editor**: See and edit your code in real-time
- **Instant Preview**: View your application as you build it
- **Built-in Terminal**: Run commands and see your app in action
- **Project Generation**: Create Next.js applications and Express.js APIs

## Quick Start

1. Clone this repository
2. Install dependencies: `npm install`
3. Set up your environment variables:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. Start the development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## How to Use

1. Visit the create page
2. Describe the application you want to build
3. The AI will generate the code and set up your project
4. Use the built-in editor to modify the code
5. See your changes in real-time in the preview window
